# HTML语法

<DOCTYPE html>是什么意思，如何回答?

一旦面试官问你一个英文是什么意思，就把他翻译成中文，所以上面的意思是文档类型。

<!DOCTYPE html>
<html >
<head>
  <meta charset="utf-8">
  <title>JS Bin</title>
  <link rel="stylesheet" href="xxx.css"/>        //这个标签只支持自己闭合,最后的斜杠没有是规范的，就是没有/的写法，那写上也对，有很强的纠错能力
</head>
<body>
 <div id=xxx>
   <div>内容</div>           //这是有内容的标签
  </div>
  <!-- <div id=xxx>
   <div>内容</div>
  </div> -->                  //这是注释
  <input type="checkbox" checked>   //这是没有内容的标签，checked=false是不看的。
</body>
</html>
