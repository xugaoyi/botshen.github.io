制作一个皮卡丘

# 制作一个皮卡丘

设计稿：

# 项目搭建

新建一个目录，比如：pikaqiu-3

创建src目录，里面创建index.html,style.css,main.js 因为项目要适配手机端，需要添加meta-vp 这里我我们直接复制淘宝的即可，打开淘宝网，开发者工具切换到手机，复制即可。 也可以直接抄我的代码：

src/index.html
<!doctype html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport"
          content="width=device-width,initial-scale=1,minimum-scale=1,maximum-scale=1,user-scalable=no,viewport-fit=cover">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>皮卡丘</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>

<script src="main.js"></script>
</body>
</html>

# HTML编写

首先搭建框架，把最外层的元素先用div框起来

<!doctype html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport"
          content="width=device-width,initial-scale=1,minimum-scale=1,maximum-scale=1,user-scalable=no,viewport-fit=cover">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>皮卡丘</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
<div class="skin">
    <div class="eye"></div>
    <div class="eye"></div>
    <div class="nose"></div>
    <div class="face"></div>
    <div class="face"></div>
    <div class="mouth"></div>
</div>
<script src="main.js"></script>
</body>
</html>

# 鼻子

鼻子是居中的，所以先做鼻子，鼻子可以转换成一个三角形和一个半圆形

# 三角形

通过定位和边框来制作三角形：

<div class="skin">
    <div class="eye"></div>
    <div class="eye"></div>
    <div class="nose">
        <div class="san"></div>
    </div>
    <div class="face"></div>
    <div class="face"></div>
    <div class="mouth"></div>
</div>
